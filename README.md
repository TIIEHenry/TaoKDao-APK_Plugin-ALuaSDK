### Usage

Clone Current Repo: https://github.com/TIIEHenry/TaoKDao-DynamicLoaderPlugin.git

clone the follow repos :

TaoKDao API: https://github.com/TIIEHenry/TaoKDao-API.git

A: use includeBuild
